# PUPPET-MOCK

Mock Puppet for Wechaty

## USAGE

```ts
import PuppetMock from '@chatie/wechaty-puppet-mock'

const wechaty = new Wechaty()

const puppet = new PuppetMock({
  profile,
  wechaty,
})
```
